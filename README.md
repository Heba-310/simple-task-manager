# React To-Do List

A simple to-do list application built with React to practice core React concepts.

## Features
- Add new tasks by typing and pressing **Enter** or clicking the **+** button
- Delete tasks individually
- Built using functional components and React Hooks (`useState`)

## What I practiced
- Managing state with `useState`
- Handling controlled/uncontrolled inputs
- List rendering with `.map()` and `key` props
- Conditional rendering
- Keyboard events (`onKeyDown`)

## Tech Stack
- React
- CSS
